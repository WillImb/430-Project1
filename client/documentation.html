Classic Literature API
API Documentation

Get Requests

    /getBook

    Docs

      Returns:

        A JSON Object containing data about the book | Format: JSON Obj

      Parameters:

        Title - The title of the book (required)

      Example

        Request: /getBook?title=book+Title
        Response: {{"author":"","country":"","language":"","link":"","pages":,"title":"bookTitle","year":,"genres":["",""]}}

    /getAll

    Docs

      Returns:

        An Array of Book Objects | Format: [JSON Objects]

      Parameters:

        No Parameters

      Example

        Request: /getAll
        Response: [{book1},{book2}, ...]

    /getTitles

    Docs

      Returns:

        An Array of Titles | Format: [String]

      Parameters:

        No Parameters

      Example

        Request: /getTitles
        Response: [title1, title2, ...]

    /getBooksByAuthor

    Docs

      Returns:

        An Array of book objects by the inputed author | Format: [JSON Objects]

      Parameters:

        author - The Author's Name (required)

      Example

        Request: /getBooksByAuthor?author=Author+Name
        Response: [{book1},{book2}, ...]

Post Requests

    /addBook

    Docs

      Returns:

        A Message stating if it was successfull or not

      Parameters:

        Title - The Book's Title (required)
        Author - The Author's Name (required)

      Example:

        Request: /addBook
        Body:{title="Title",author="Author"}
        Result{message:"Book Added"}

        Request: /addBook
        Body:{title="Title",author=""}
        Result{error:"One or more parameters missing"}

    /addRead

    Docs

      Returns:

        A Message stating if the status was successfully updated

      Parameters:

        Title - The Book's Title (required)
        Status - The User's read status of the book (required)

      Example:

        Request: /addRead
        Body:{title="Title",status="read"}
        Result{message:"Book Read Status Updated"}

        Request: /addBook
        Body:{title="",status="unread"}
        Result{error:"No Title Entered"}
